OHIF ZIP Exporter

A custom OHIF viewer build with support for exporting .zip files.

Features:

- Adds an extension which defines functionality for an "Export ZIP" button.

- Adds a mode which adds the "Export ZIP" button to the viewer's toolbar.

- Button exports a .zip file including patient metadata and selected viewport image.

TeXQuick

A lightweight, LaTeX-powered web app for mathematical note-taking.

Features:

- Notes are persistently stored in localStorage between sessions.

- Notes can be edited and deleted at will.

- Multiple notepads can be managed simultaneously.

- Notepads can be exported and imported as .txt files.

Birds of Aotearoa

A full-stack web app for managing a database of New Zealand birds.

Features:

- RESTful Express.js routing supports a full range of HTTP requests.

- Renders server-side HTML dynamically using EJS templates and partials.

- Features support for all of the standard CRUD operations on a MySQL database.

- Fully containerised with Docker for cross-platform deployment.
