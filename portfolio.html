NZ Bird Browser

A web app to allow for querying and favouriting New Zealand birds.

Features:

- Allows users to query birds based on name and conservation status.

- Retrieves JSON data via JavaScript's Fetch API and adds corresponding elements to DOM.

- Normalises search queries to NFC to account for both English and Te Reo MƒÅori names.

- Allows for favouriting of birds in persistent local storage, to retrieve at a later date.

- Has both blue and red themes which can be switched between.
